# Buckshot Roulette - 血腥版俄羅斯輪盤遊戲

一款基於心理戰的恐怖生存遊戲，玩家與AI輪流使用霰彈槍玩俄羅斯輪盤。

## 遊戲規則
- 三局制，每局子彈數量增加
- 多種道具可以改變遊戲進程
- 第一、二局死亡會被除顫器救活
- 第三局是真實死亡

## 道具系統
- **香菸**：恢復1點生命
- **手銬**：跳過對方下一回合
- **手鋸**：下一發子彈傷害翻倍
- **啤酒**：移除當前槍膛中的一顆子彈
- **放大鏡**：查看下一發子彈是實彈還是空包彈
